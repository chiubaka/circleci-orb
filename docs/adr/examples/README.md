# ADR Pattern Examples

These examples are repository-local illustrations that complement org-level ADRs.

Org ADRs stay portable and normative; these files show one way the patterns can appear in this repository.

- `feature-module-layout-example.md` — vertical feature slice with layer and feature barrels.
- `composition-root-example.md` — edge composition root and thin HTTP host shape.
- `import-aliases-example.md` — package-local alias usage for `~/` and `#/`.
- `small-focused-files-example.md` — when to split files vs keep related contracts together.
